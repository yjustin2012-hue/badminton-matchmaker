# Badminton Matchmaker

An iPad-first PWA for fair badminton session management, match generation, live rankings, and scoring.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Features

- Fair match generation with player rotation
- Live rankings and statistics
- Session snapshots and roster presets
- Due-up indicators for fairness
- Quick undo for recent results
- Three language support (English, Simplified Chinese, Traditional Chinese)
- PWA support (offline capable)

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Dexie (IndexedDB)
- i18next (localization)
- Vite PWA
